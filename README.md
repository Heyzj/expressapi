# expressapi
### 这是一个基于express web框架开发的后台管理接口api
---  
### 接口说明：https://www.showdoc.com.cn/escook?page_id=3707158761215217
--- 

##### 1.初始化项目 

`npm init -y`

##### 2.安装依赖包 

`npm install`

##### 3.运行服务 

`node app.js`

###### 关于使用该API的前端源码在：https://github.com/Heyzj/bigdata ，使用时需要将请求根路径更换wei `http://127.0.0.1`，数据库使用Mysql ,在文件 **/db** 下
